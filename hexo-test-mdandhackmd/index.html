Markdown 顯示語法測試

2017-07-11 | 2018-03-29 | computer |
| 42k

這裡是從Hackmd的官方介紹中直接把原始碼拿過來用的，並作解析MarkDown語法測試。這裡也有完整中文介紹與維基百科介紹。這個語言的重點在於方便直接看懂內容，並且容易學習。相較於沒有任何標示的純文字文件而言，MarkDown的確是一個可以快速幫你標記，並且紀錄各種想法的好幫手。另一方面，龐大無比功能複雜的word絕對不是一個紀錄筆記的好方式（當然也可以選擇用word…但若要把word的內容直接發布到網誌上也是需要花費一番功夫的。）那為什麼不選擇在許多部落格都支援的Markdown語法作為寫作的最佳幫手呢？當然，本站主要以Hexo+Github在fedora27作業系統下架設。
