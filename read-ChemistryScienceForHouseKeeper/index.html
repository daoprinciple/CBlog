家事的科學-生活上的清潔劑選用指南

2020-07-27 | reading |
| 2.3k

市面上清潔物品百百種，要找到一款適合全部用途的清潔劑有多難？而這種萬用清潔劑是不是真的存在？想想衣服在不同的場合都需要穿不一樣的衣服跟搭配了，更何況是百百種來源的髒污呢？而這次要介紹與節錄的書家事的科學就清楚點出各種生活場合常用的清潔劑較為推薦與合適的成份。

這本書適合對象為一般有些國中理化的基礎底子的人。裡面介紹到很多化學成份的結構式名稱，當然，他的化學機制只講最基本的，並沒有深入探討化學作用的機轉原理（其實也不需要）。如果有興趣更近一步了解，這本書內容可能不足，但就以生活上清潔劑選擇來說，已經足夠。而那些令人看不懂毛骨悚然的一大串英文（或中文名）其實在書中都有附上基本的相對應概念，例如去污力強弱，傷肌膚程度，
中英文名對照等等。如果想要更近一步照顧自己的身體與居家環境，花點時間去理解自己所用的清潔用品是個值得投資的知識點。

¶重要概念

  • 天然的一定好嗎？
    化工製品相對單純，天然反而有機會造成各種身體不適。舉個例子，有人討厭吃天然的香菜，但應該沒有人討厭吃加工的巧克力糖果餅乾等等。

  • 清潔原則

    1. 油漬 -> 界面活性劑（有分陰/陽/雙）
    2. 髒污表面傳統上需要加上清潔劑洗刷才能從表面帶走
    3. 酸鹼平衡

  • 會與肌膚接觸選低刺激性

  • 帶正負靜電的物品可用其相對的靜電吸附方式來去除髒污，例如毛髮。

¶各種常用清潔品

以下列出幾個個人比較常用的清潔用品，並且附上主要成份的用途與書中（或是我自己）的選擇策略。

¶身體清潔

  • 牙膏

    1. 研磨劑：夠硬的研磨劑就好，不然就是在磨損牙齒（約6-7莫氏硬度）
    2. 氟化物：預防蛀牙。常見的牙膏品牌幾乎都含有此成份。
    3. (未知)磷酸鈉/焦磷酸鈉：預防結石
    4. (未知) 葡聚糖：預防牙菌斑

  • 沐浴乳

    1. 椰油醯谷氨酸鉀 Sodiumlauroylglutamate
    2. 挑選成分單純，給寶寶用的其實不錯。

  • 洗髮精

    1. 盡量不選用月桂醇硫酸脂鈉 強效清潔
    2. 月桂醇聚醚-4羧酸比較沒那麼強 堪用
    3. 矽靈沒那麼恐怖，但好的洗髮精的確也不會加入（？

¶用具的清潔

  • 浴廁廚房

    1. 去黴劑：鹼性+次氯酸鈉（注意不可以跟鹽酸混合！！！！！！！）
    2. 水垢/水銹（白斑）尿垢：檸檬酸/弱酸性/嵌合劑（？for金屬微粒
    3. 油脂污垢：雙介面活性劑/鹼性劑/碳酸氫二鈉

  • 洗碗精

    1. 可選擇不傷手成份
    2. 你以為的小蘇打其實不太好去污

¶衣物洗滌

  • 分為陰離子/石鹼/雙離子，一般來說冷洗精為雙離子或非離子洗衣精，比較好用。
  • 洗衣精跟洗衣粉相比，粉狀溶於水與液態相比，液態應該較為容易混合均勻。
  • 冷洗精無法除臭與抗菌，可以適時漂白與烘乾。

¶願望清單

其實極簡主義不應該有這個念頭才是，事情應該要做簡單而並非複雜。因為這個社會越來越複雜與分工越細，各種專門的去除令人討厭味道與去污的產品相繼而生，所以在這寫下一些我認為可以購買但並非必要的可能產品。也許擁有這些會讓你的人生變得更舒適，但不代表就一定會舒適。別忘了，物品越多越會造就保養與維護的成本越來越高。

  1. 過濾蓮蓬頭
    過濾氯，對皮膚產生刺激性的物品有可能是來自於氯。
  2. 無香空間
    討厭氣味的人可以買這個去除雜味，我個人用了空氣清淨機+除濕機，應該不需要買這個了哈。
  3. 馬桶清潔錠
    這個感覺很方便，但還不確定是不是真的需要，就每天洗完澡稍微刷一下就好拉。
