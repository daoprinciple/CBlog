持續的技術

2019-02-10 | 閱讀 |
| 2.3k

¶核心概念[1]

  1. 認知當下正在做的事情與心情（做紀錄），可以用模塊化的方式切割（分類理解，最後再整合）。
  2. 從小事到大事的事件（物件）導向處理能力慢慢建立。
  3. 利用空間（場所）/時間（生理時鐘）/動作（儀式感）/言語（簡單明確的指令）使身體記憶自己的持續能力，並非是使用意志力支撐，意志力只有三分鐘熱度。

¶戒斷 -認知當下

認知自己正在做什麼的能力，是為腦內在不正常的產生大量多巴胺的結果。一直在做的事情不代表是真正想做的事（錯誤的獎勵機制）。

  1. 認知自己為何被觸發想要做的狀態，可以搭配冥想加強對自己的認知能力。
  2. 做任何事情前先預想為什麼自己要做這件事
  3. 因為時間守恆-> 每日能做的選擇數量是有限多的-> 減少所需做決定的次數與選擇的內容

¶內在動機-自己決定

回復自己選擇的能力，而不是依照外在給你的感覺選擇。說到底這還是很困難，在接觸越來越多事務之後，很難說出自己到底喜歡什麼樣的東西。而在操作上我覺得相對之下較為困難，只能從小地方開始著手慢慢改變自己的做事習慣。

  • 改成晨間日記，不是反省過去而是預備今天，也就是活在今天，掌握今天該做的事物。
  • 有意識的做任何決定，不論是午餐吃什麼，甚至是等一下要從什麼事情開始做起，要讓腦袋回到自己做決定的時代而非行屍走肉般的自動模式。這有助於斷離自己做出令自己後悔的決定。
  • 拋開腦中的偶像心理，把世界上的偉人們都丟掉吧，你不是他們也永遠不會是他們，請專注於自己而並非他人的成就。

¶持續

有個被動影響的方式，也就是物理學上最愛探討的議題：狀態隨時空間的變化。我們可以藉由訓練自己在固定的空間/時間/本身的動作/本身的言語來造就自己心中理想的生活模式。

  1. 空間：在合適的時間點做合適的事情，一個地點只做一件事情。e.g. 床就是睡覺的地方，書桌就是讀書，耍廢角就是在耍廢。因此到了該地就會自動讓身體沈浸在過去記憶的氛圍裡。
  2. 時間：以每週（每日）為單位，在固定的時間點做固定的事情。持續不用連續，容許自己有失敗與修改的權利，才會進步的快。
  3. 動作：在合適的身體動作做合適的事情。例如運動前會先熱身讓自己的身體進入工作狀態（暗示）。平常做事情前也應該有如此的儀式感，做一些讓自己身體放鬆的動作，並且在腦中暗示自己要開始做事了，很快就會進入心流的狀態。
  4. 言語：外在物理上的作用（動作）外，心理暗示也是使自己進入狀態的重要一環。「次數+時間+工作量」具體的表達出來，有助於在時間上達到自己的目標。當然這估計能力是需要長期累積下來才能完成的。

¶不中斷的持續& 滿足需求

這後面的技術我認為較為高階，也就是滿足了上面所說的狀態後，繼續用觀察自己的當下狀態（心情,喜好的事物），進而持續地改善自己的狀態。而最好的方式便是紀錄狀態，這樣才能從過去推向現在進而改變自己。而思考時並非用單一點的方式去思考現在在做什麼，而是利用整體的人生概念去思考現在為什麼要做這件事情。例如洗澡時間，購物時間，讀書時間，工作時間，與家人做事時間等等。
