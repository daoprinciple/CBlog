該如何做一場TEDTalk?

2017-11-19 | 閱讀 |
| 5.1k

¶這本書有什麼有趣的嗎？

看過很多TED演講後，發覺TED真的不錯。基本上，演講內容都很值得去思考與反思關於自己、關於朋友家人、關於環境、還有關於這個世界究竟是怎麼一回事。而這些有趣的觀點，或是他人想傳遞的觀點，藉由一場場18分鐘的演講，傳遞給世界上各地的人。又因為藉由網路的緣故，已不再像古代的資訊傳遞如此不便，只要是能夠上網的地方，就能夠獲得最新的演講內容，不論是否在地球的另一端。

但是上面說明還是不足以說服自己為什麼要看這個？最重要的是想知道，其他人在分享資訊時所花費的經歷究竟是放在何處？為什麼他們都可以在台上給這麼精彩的演講？但實際上，並不是所有人都有辦法可以完成這18分鐘的講演，不僅僅只是背槁的準備這麼簡單，還必須要全面性的討論一個人的準備方式是否完善。這本書深入淺出的講解一般而言的演講方法，並且細部介紹各種演講工具、如何準備、在台上演講的注意事項、以及對於這整個TED活動的反思。下面所寫出的內容，有經過個人偏好篩選與整理，如果真的對這本書有興趣的朋友，可以到各書局找找看。

¶演講的重點

相較於書籍而言，一場簡潔有力的演講可以用淺顯易懂的語言，面對面的利用口語、肢體表達、投影片的方式將想法散佈出去。是一種雖然速度比較慢，卻印象較為深刻的散佈方式。個人認為一場演講要成功，是要能夠有效的傳遞你的核心價值給觀眾並且讓觀眾能夠接受。因此，好的演說表達能力，與核心價值觀的傳遞，在整個演講中，佔有舉足輕重的地位。

而書中提到四點容易犯下錯誤的演講

  • 推銷
    除非是要跟客戶做簡報或者應徵，否則你應該要推銷的不是組織，而是想法。
  • 漫談
    講到廢話去了，偏離核心思想的隨便聊天。講太多無意義的細節。
  • 組織
    沒人想知道寫在組織章程裡的條文，也沒人會想知道組織的歷史，除非你的主題就是為什麼組織會興起。當然，這也要等到你超級有聲望他人才會想知道你這個組織究竟是怎麼建立起來的。
  • 為了鼓舞而鼓舞
    講幹話的人最愛，在台上為了獲得觀眾的目光，用了很多很炫的演講技巧，並且吸引觀眾的目光，但沒有核心思想。

¶演講工具

主要有五種，而依照演講會依序出現的順序列出，因此在準備一場演講時可以同時利用這五個技巧。但這些技能不代表全部都要用上，可以選擇性的用上。

¶Connection 連結個勾勾

演講跟文字表達寫書不相同，必須要有對話的方式才能夠勾引他人對於演講者的興趣。當然他們會想來聽演講是因為對這方面的內容本身就有興趣了，但講者也必須要在大家還沒有熱情消退之前，對聽眾勾引起對於這主題的好奇心。這樣在後面討論更為深入的主題時，才能夠讓聽眾進入狀況。作者在書中提到了三個方式，而以下列出兩點:

  • 眼神接觸
    講話都要眼神接觸了，講演講當然也要阿！而且如果有帥哥/妹子，還可以正大光明跟對方眼神交換，好機會，不接觸嗎XD
  • 展現脆弱
    人內建銅鋰鋅（不要跟我戰為什麼有，基本上正常人都有），可以自嘲自己的緊張，畢竟觀眾也是知道上台需要很大的勇氣與準備的。所以不吝嗇的將自己脆弱的一面展現出來吧！說不定會因此感到放鬆而演說的更好也不一定

¶Narration 說個小故事

故事是一個傳遞訊息的好方法。老實說，我認為整個演講，或者說傳遞想法，本身就是一種說故事的方式。而其他人能否接受你說的故事，就端看故事內容有不有趣。在演講過程中，說小故事或者寓言來輔佐整篇的大主題是一個不錯的選擇。下面有四點有關一個好故事應該要有的內容

  1. 能夠使觀眾產生移情作用的人物
  2. 透過好奇心、社會情節、或真實的危險情寄來營造張力
  3. 提供適量細節，細節太少，故事就不生動。細節太多，故事就不流暢。
  4. 有個令人滿意的結局，像是有趣、或感人，或有啟示的含意。（說故事是要有目的的）

¶Explanation 清楚的解釋

簡單清晰淺顯易懂，講給你老媽聽也讓他熱淚盈眶。理解一個概念跟能夠將這個概念說給其他人聽是兩回事。雖然說理解一件事物，但很容易會掉入自己知道或者同行知道你在說什麼，他人卻完全無法理解你在說什麼的窘境。因此利用淺顯易懂的言語將困難的話語說給他人聽需要不斷練習與重新思考如何才能講的更為清晰與簡單。

¶Persuation 有利的說服

利用各種證據與脈絡，抽絲剝繭的將研究結果或是人生中獲得的感想傳遞給他人時，會有比較強力的說服力。而這個在科學訓練上通常都較為充足。也就是說如果受過了科學訓練卻無法學會如何說服他人理解自己所想的，或是站在他人立場思考，那這部份可能要看相關的邏輯書或者是經由日常訓練去理解一件事物的前因後果。

¶Revelation 翻開黑布的揭露

這就是展示，沒啥好說的。有時候直接實做給他人看，遠比於說的天花亂墜要來的有用的多。因此如果有一些有趣的物品道具、或是影片展示、成品展示，都是個很棒的機會可以直接將概念傳遞給他人的方式。

¶如何準備一場演講

¶視覺

除了人到場的演講外、有時候也會輔以圖片、影片、投影片的方式。而這投影片的演講內容會發布在另外一份文件中(或者日後在此篇文章中做修改)。這在之前的上台報告中花費了一些經歷去做練習，所以對於這部份有些感觸，就不會完全照著書中所寫的大綱，而是會加入一些以往的經驗詳細的寫下。這對於以後自己再做投影片時也能夠當作參考資料。

¶腳本

有兩派，背槁、與不背槁的準備。背槁需要寫一篇完整的稿子，從頭到尾一字不漏的背起來，優點是使人放心，缺點如果是忘記的話很容易會緊張而整個GG。另一方面，如果是準備大綱式的演講，也是需要大量練習的。自己需要做過很多次的練習，對其他人演講的練習，才有辦法真正上台去講給其他人聽。而這種練習的好處是可以掌握整個大節奏，也可以以較為口語的方式將自己的想法利用口語表達出來。而很明顯的缺點是，如果是較為專業的部份，很容易會掉入自己的細節舒適圈中而不自知。個人比較喜歡大綱式的準備方式，能夠在台上比較迅速的應變任何的問題。

¶預演

練習讓世界更美好，歐其實是練習會讓事情變得越來越簡單。當然要相信這是真的！多練習，多接觸，可以讓自己在某方面的能力上升。這不是必然，但在演講準備的過程裡，練習同一個主題十幾次不是鮮少的事情。很多TED演講人都會在上台前做過N次的練習，也會請他人幫忙校稿。所以不要以為他們一開始就可以上台給個專業的演講，這是不可能的。沒有練習，沒有可能會熟練一項技巧。

¶開場/結尾

這重要性不必說明惹，總之就是很重要就是了（被打）。而各種工具的使用在前面就已提過，這裡不再贅述。

¶歐！我上台了該注意什麼

¶身

身體狀態、外表是否適合上台？在上台前幾天就應該了解自己上台要給他人的形象是什麼，而不是上台前一小時還在準備自己的行頭。這非常的要命。所有的事情都應該要事先準備，才能夠達到比較好的展現成果。

¶心

台上最容易的是忘記自己接下來要說什麼，而要減緩這種忘記的方式，除了多練習外。還有自己寫個備忘錄、小抄等等上台也是個不錯的選擇。不需要真的去看小抄，演練的過程不需要看小抄，小抄是避免自己忘記的好方法。而通常這張小抄幾乎都不會用到，但如果這對上台演講的心情有幫助的話，那就抄吧。

¶環境與工具

有沒有想要改變自己的演講方式？有時候有些特別的演講可以吸引他人的目光，但是這必須要慎用。如果用過頭華麗的技巧，那他人只會街收到技巧而並非演講的主題內容，這反而是喧賓奪主。而事先跟主辦單位或者演講場地的觀看也是非常重要的事情，如果沒有提前去做準備，常常會發生技術性的困難以及各種對於場地不熟悉而引起的尷尬、不舒服、忘詞、緊張。

¶讀後的反思

看完本書，總是對於內容有些疑惑，雖然TED號稱是可以分享任何值得分享的想法，但這只建立在其主流價值觀上。很多非主流的想法、政治不正確、怪力亂神的內容並沒有被包含在裡頭，雖然說我不確定鑽研這方面研究的人（對這方面有興趣的人）是不是整體而言沒一個會講話的，但如果能夠在TED演講中提供這方面的想法，也許才能真正的包容全部的思想才是。
此外，其實現在新興的youtuber行業，雖然是基於影音平台去做這樣的事物，但已經逐漸成為主流年輕人分享資訊的地方。TED也許是這方面的發起，而且在youtube上放置了大量值得深思的演講影片在其中，但這樣的演講還是無法取代真正去現場感受那種演講的氣氛。如果可以，還是希望能夠到現場（以後的VR技術也許可以打破這種感覺，讓所有人在家中就能夠體驗在現場的感受），去感受講者想表現出的一切。當然有了網路，才能夠快速的將各種想法資訊，經由網路快速的傳遞到世界各地去。
