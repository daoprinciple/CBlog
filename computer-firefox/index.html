瀏覽器推薦 firefox(火狐)

2017-09-13 | computer |
| 3.3k

¶火狐是一款什麼樣子的瀏覽器？

Firefox是由Mozilla開發的瀏覽器。這個非營利組織大約在1998年左右從Netscape中脫離出來（這是一款已經死掉的瀏覽器…現在應該沒有什麼人在使用了吧。）而後來Mozilla社群秉持著自由軟體社群該有的風氣，開始了Firefox瀏覽器的開發。基於Mozilla的十點宣言，主要可以想成下列幾點開放、易於取用、個人隱私、平衡商業與公眾利益。 雖然以前我也棄坑跑去使用google chrome，但基於對於個人的思想理念，認為開源、知識共享是一件很重要且必要的事物，並且Firefox有對於個人隱私強大且細部的設定，能讓使用者在短時間內解決各種疑難雜症。最重要的效能方面，現在改版過後的火狐啟動速度與chrome相較之下，已經沒有優劣之分（甚至有時候的更新版本會比調色盤快速）。當然如果你需要許多阿里佈達的東西，也可以藉由火狐的各種擴充套件增強（對，瀏覽器的各種擴充套件如果我沒有記錯是從火狐開始的）

¶為什麼要選用火狐呢？

  1. 促使chrome進步
    其實我覺得這個理由有點瞎，但是可以想想看以前的IE到底有沒有在進步XD。如果少了競爭者而獨大，很容易會不思進取（當然我相信chrome已經開始在走下坡了XD，不知道為什麼它已經沒有當初宣稱的如此開啟快速，歐，其實應該是因為加了太多的附加元件而使得自己所使用的chrome越跑越慢）。而且根據StatCounter的瀏覽器使用者人數評比(雖然有灌水的疑慮，也有低估的疑慮，但使用者大致上人數是差不多的)，2017年八月的Chrome市占率高達54％，剩下的才是其他家分，而火狐只剩下了5.9%有興趣的話可以看一下Register的文章-Why Firefox? Because not everybody is a web designer, silly

  1. 為了自由、隱私
    Chrome會為了進步而收集你的個人資料，當然你會在糊里糊塗的情況就勾選了。想要取消也不是不行。不過在搜尋以及使用者體驗上，chrome的確是因為收集你的個人資料，而使得你的搜尋會更加的客製化，想當然爾，你也會看到更多你想要看到的資料，就是如同臉書的網路同溫層效應（也因此現在會越來越常常幫跟自己立場相反的人點讚，這樣才能看到跟自己想法不同人的聲音。）這件事情重要嗎？見仁見智，如果不是很在意自己的資料外流的話，其實是沒差，但如果你很在意，那就改用火狐吧！

  2. 求新
    如果說對於最新的功能有喜歡的話，可以使用開發中的版本- Firefox-Nightly。目前使用來看，他有長期開啟之後會佔用越來越多CPU的樣子，但我在想應該是因為他是開發板，也因此有很多有趣的內建功能，大概可以提前三個月先體驗看看好用的功能（例如container，可以直接在分頁中開啟同一個網站中不同的帳號。如果你要用好幾個FB帳號撩妹，我想用這個是非常棒的功能。喔說錯了，是說可以把一些個人隱私的部份跟公司、或者是購物的部份做區別）。

  3. 有趣的功能- 收藏庫
    除了我的最愛（書籤）以外，如果碰到有趣的文章想要晚點再看的話，使用火狐內建的收藏庫可以有效率的先將文章暫存起來並且關掉，等到有空的時候再來讀，這樣不僅可以讓你專注在目前需要做的事情上，也可以讓你不因為一直掛著那個吸引你的分頁而分心。另外有趣的事情是，經過一小段時間的沈澱，你可能對於為什麼要看這篇文章感到茫然，並且問自己說，為什麼自己要花時間看這篇文章呢？因此而直接將網頁關閉而不點進去看。個人的經驗是，網路上資訊太多，太容易取得很多廢料、廢料、跟似是而非的廢料。根據上述三點，便捷、暫時關閉、給予自己再次思考是否要看這篇文章的機會
    ，這個功能對於網路重度使用者或者網路工作者是一個十分實用的工具。

  4. 更多有趣的功能
    當然你可能會說，為什麼不要用chrome?它還有很多人在維持的附加元件阿！為什麼不要裝一堆好用的附加元件就好?就我所知，你加了很多附加元件會拖慢瀏覽器本身的速度，因此而讓整個瀏覽器變成吃資源的怪獸（以前的火狐就是因為這樣才讓很多人汽坑）而這些有趣的功能可以自己去找尋摟！（覺得sync還滿好用的，可以記憶你的火狐各種設定在不同平台的瀏覽氣上，舉個例子如果你在上廁所時使用手機看到有趣的文章
    ，雖然廁所上完了但文章還沒有看完，這時候可以用sync傳送到電腦。）

¶有什麼有趣的add-on（附加元件）?

  • 新同文堂
    這應該是使用中文的人必備的軟體，可以快速的在繁體-簡體之間做轉換，看小說必備XD。

  • Owl
    這款add-on主要是用在閱讀，如果習慣了黑底的舒適閱讀環境，這款附加元件能夠輕鬆的讓你在舒適的環境中閱讀文件。

  • Privacy Badger（隱私獾）
    主要功能是為了讓你的網路瀏覽更具有隱私。而他的記憶體與CPU使用率也是低消耗。但這款addon並不是為了要阻擋各種廣告，所以如果你需要阻擋廣告的話還要另外一個附加元件- uBlock Origin

  • uBlock Origin
    一款自由、開源、跨平台的內容過濾瀏覽器擴充功能，包含廣告遮蔽功能。就如維基百科所敘述的一樣，他可以強化你的網頁閱讀體驗，並且將很多不必要的廣告過濾掉，專心在自己所想要看的內容上（想想看被整治過後的台灣市容一點廣告也不負存在也沒有五顏六色各種沒設計感的招牌，是多麼令人舒適的一件事情！）
